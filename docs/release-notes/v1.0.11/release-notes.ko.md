# CodexPoolManager v1.0.11

릴리스 날짜: 2026-06-08

## 주요 변경 사항

- API key를 사용하는 릴레이 provider로 Codex CLI를 수동 전환할 수 있는 API Key 릴레이 계정을 추가했습니다.
- 인증 흐름을 OAuth / 구독 계정과 API Key 릴레이 계정의 두 경로로 나누어 더 명확하게 만들었습니다.
- Codex의 기존 기록을 계속 표시하면서 API 요청을 릴레이 Base URL로 보내는 기록 유지 모드를 추가했습니다.
- 릴레이 계정에서 구독 계정으로 돌아갈 때 OAuth 인증 정보와 provider 설정이 깨끗하게 복원되도록 수정했습니다.
- API key 릴레이 계정은 계정 목록의 마지막에 배치되며, 사용량 동기화와 자동 전환 대상에서 제외됩니다.
- 릴레이 설정 폼을 개선했습니다. Base URL은 주요 필수 입력란으로 이동했고, API 형식 설명을 추가했으며, Base URL 기본값은 비워 두었습니다.
- 릴레이 계정 UI와 릴리스 노트의 다국어 지원을 보강했습니다.

## 참고 사항

- API key 릴레이 계정은 ChatGPT 구독 사용량 데이터를 제공하지 않으므로 수동 전환만 지원합니다.
- 기록 유지 모드는 다음에 릴레이 계정으로 전환할 때 적용됩니다.
- 수동 마이그레이션은 필요하지 않습니다.
