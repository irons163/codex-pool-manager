# CodexPoolManager v1.0.13

Date de publication : 2026-06-09

## Corrections

- Correction d’un crash possible lors du passage à un compte relais API key depuis le panneau d’utilisation des comptes.
- Mise à jour de la version de l’app et du build number afin que Sparkle puisse proposer correctement cette hotfix après v1.0.12.

## Notes

- Aucune migration de compte, API key, auth.json ou config.toml n’est requise.
- Cette hotfix est recommandée pour les utilisateurs de v1.0.12 qui utilisent des comptes relais API key.
