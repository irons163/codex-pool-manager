# CodexPoolManager v1.0.13

リリース日：2026-06-09

## 修正

- アカウント使用量パネルからリレー API key アカウントへ切り替える際に、クラッシュする可能性がある問題を修正しました。
- Sparkle が v1.0.12 以降の hotfix として正しく検出できるよう、App のバージョンと build number を更新しました。

## 注意事項

- アカウント、API key、auth.json、config.toml の移行作業は不要です。
- リレー API key アカウントを使用している v1.0.12 ユーザーに、この hotfix への更新をおすすめします。
