# CodexPoolManager v1.0.13

Fecha de publicación: 2026-06-09

## Correcciones

- Se corrigió un crash que podía ocurrir al cambiar a una cuenta relay API key desde el panel de uso de cuentas.
- Se actualizó la versión de la app y el build number para que Sparkle pueda ofrecer correctamente esta hotfix después de v1.0.12.

## Notas

- No se requiere migrar cuentas, API key, auth.json ni config.toml.
- Se recomienda esta hotfix para los usuarios de v1.0.12 que usan cuentas relay API key.
