# CodexPoolManager v1.0.13

Release date: 2026-06-09

## Fixes

- Fixed a crash that could happen when switching to a relay API key account from the account usage panel.
- Updated the app version and build number so Sparkle can correctly offer this hotfix after v1.0.12.

## Notes

- No account, API key, auth.json, or config.toml migration is required.
- This hotfix is recommended for all v1.0.12 users who use relay API key accounts.
