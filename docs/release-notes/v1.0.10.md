# CodexPoolManager v1.0.10

Release date: 2026-05-21

## Highlights

- Fixed an issue where canceling OAuth sign-in could leave the flow hanging.
- Fixed usage analytics deduplication across groups and improved paid-account prioritization.
- Restored expected account usage panel height inside nested scroll layouts.
- Prevented stored API tokens from being cleared when an incoming snapshot has an empty token field.
- Improved dashboard and auth flow reliability with broader coordinator and smoke test coverage.
- Added API Key / Relay provider accounts for Codex CLI custom provider switching. Relay accounts use `codex login --with-api-key` and are manual-switch only because ChatGPT subscription usage sync does not apply to API-key relay credentials.

## Notes

- No migration steps required.
- DMG artifacts are published for both Apple Silicon and Intel builds.
