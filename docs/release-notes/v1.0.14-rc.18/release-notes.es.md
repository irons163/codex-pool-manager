# CodexPoolManager v1.0.14-rc.18

Fecha de lanzamiento: 2026-07-02

## Mejoras

- Las tarjetas de cuenta de la ventana principal ahora muestran créditos de restablecimiento disponibles.
- En modo normal, cada tarjeta lista el vencimiento estimado de cada crédito de restablecimiento.
- En modo `Minimal`, las tarjetas resumen los créditos con texto compacto como `2 restablecimientos · 7/30, 8/1`.
- Se añadió un aviso `Novedades` localizado tras cambios de versión/build, con una entrada en Settings para volver a abrir las últimas notas.
- El aviso de novedades está localizado en todos los idiomas soportados por la app.
- La documentación README se actualizó en todos los idiomas soportados para explicar la visualización de créditos y el aviso de novedades.

## Notas

- El vencimiento de los restablecimientos se estima sumando 30 días a la sincronización correcta anterior; el vencimiento real puede variar.
- Esta prerelease sirve para validar la visualización de créditos de restablecimiento en la ventana principal y los avisos de novedades localizados antes del lanzamiento stable.
