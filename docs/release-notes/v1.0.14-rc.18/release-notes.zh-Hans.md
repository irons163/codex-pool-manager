# CodexPoolManager v1.0.14-rc.18

发布日期：2026-07-02

## 改进

- 主窗口账号卡新增可重置额度信息。
- 普通模式账号卡会列出每一次可重置额度的推估期限。
- `极简` 账号卡会以 `可重置 2 次 · 7/30, 8/1` 这类精简日期文字显示。
- 版本或 build 变化后新增本地化“新功能”提示，并可从设置重新查看最新功能说明。
- 新功能提示已补齐所有 App 内置语言。
- README 已同步更新所有支持语言，说明可重置额度显示与新功能提示。

## 注意事项

- 可重置额度期限是根据上次成功同步时间加 30 天估算，实际期限可能不同。
- 此 prerelease 用于正式发布前验证主窗口可重置额度显示与多语言新功能提示。
