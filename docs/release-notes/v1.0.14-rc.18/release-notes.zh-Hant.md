# CodexPoolManager v1.0.14-rc.18

發布日期：2026-07-02

## 改進

- 主視窗帳號卡新增可重置額度資訊。
- 一般模式帳號卡會列出每一次可重置額度的推估期限。
- `極簡` 帳號卡會以 `可重置 2 次 · 7/30, 8/1` 這類精簡日期文字顯示。
- 版本或 build 變更後新增本地化「新功能」提示，並可從設定重新查看最新功能說明。
- 新功能提示已補齊所有 App 內建語系。
- README 已同步更新所有支援語系，說明可重置額度顯示與新功能提示。

## 注意事項

- 可重置額度期限是依前次成功同步時間加 30 天推估，實際期限可能不同。
- 此 prerelease 用於正式發布前驗證主視窗可重置額度顯示與多語新功能提示。
