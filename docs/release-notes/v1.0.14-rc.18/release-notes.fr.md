# CodexPoolManager v1.0.14-rc.18

Date de publication : 2026-07-02

## Améliorations

- Les cartes de compte de la fenêtre principale affichent désormais les crédits de réinitialisation disponibles.
- En mode normal, chaque carte liste l'expiration estimée de chaque crédit de réinitialisation.
- En mode `Minimal`, les cartes résument les crédits avec un texte compact comme `2 réinitialisations · 7/30, 8/1`.
- Ajout d'un prompt `Nouveautés` localisé après un changement de version/build, avec une entrée Settings pour rouvrir les dernières notes.
- Le prompt de nouveautés est maintenant localisé dans toutes les langues supportées par l'app.
- La documentation README a été mise à jour dans toutes les langues supportées pour décrire l'affichage des crédits et le prompt de nouveautés.

## Notes

- L'expiration des réinitialisations est estimée en ajoutant 30 jours à la synchronisation réussie précédente ; l'expiration réelle peut différer.
- Cette prerelease sert à valider l'affichage des crédits de réinitialisation dans la fenêtre principale et les prompts de nouveautés localisés avant le déploiement stable.
