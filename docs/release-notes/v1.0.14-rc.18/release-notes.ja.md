# CodexPoolManager v1.0.14-rc.18

リリース日：2026-07-02

## 改善

- メインウィンドウのアカウントカードに、利用可能なリセット情報を追加しました。
- 通常モードのアカウントカードでは、各リセット枠の推定期限を一覧表示します。
- `Minimal` のアカウントカードでは、`リセット 2回 · 7/30, 8/1` のような短い日付表示にまとめます。
- version/build の変更後にローカライズされた「新機能」案内を表示し、Settings から最新機能ノートを再表示できるようにしました。
- 新機能案内を、すべての対応アプリ言語に追加しました。
- README をすべての対応言語で更新し、リセット情報表示と新機能案内を説明しました。

## 注意事項

- リセット枠の期限は前回の同期成功時刻に 30 日を加えて推定しているため、実際の期限とは異なる場合があります。
- この prerelease は、stable 公開前にメインウィンドウのリセット情報表示と多言語の新機能案内を検証するためのものです。
