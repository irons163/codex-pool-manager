# CodexPoolManager v1.0.14-rc.18

릴리스 날짜: 2026-07-02

## 개선 사항

- 메인 창 계정 카드에 사용 가능한 초기화 정보를 추가했습니다.
- 일반 모드 계정 카드는 각 초기화 크레딧의 예상 만료 시각을 나열합니다.
- `Minimal` 계정 카드는 `초기화 2회 · 7/30, 8/1`처럼 간단한 날짜 텍스트로 요약합니다.
- version/build 변경 후 현지화된 “새로운 기능” 안내를 표시하고, Settings에서 최신 기능 노트를 다시 열 수 있게 했습니다.
- 새로운 기능 안내를 모든 지원 앱 언어에 추가했습니다.
- README를 모든 지원 언어로 업데이트해 초기화 정보 표시와 새로운 기능 안내를 설명했습니다.

## 참고

- 초기화 크레딧 만료는 이전 동기화 성공 시각에 30일을 더해 추정하므로 실제 만료 시각과 다를 수 있습니다.
- 이 prerelease는 stable 배포 전에 메인 창 초기화 정보 표시와 다국어 새로운 기능 안내를 검증하기 위한 버전입니다.
