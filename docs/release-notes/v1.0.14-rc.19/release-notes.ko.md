# CodexPoolManager v1.0.14-rc.19

릴리스 날짜: 2026-07-03

## 개선 사항

- README의 메뉴바 스크린샷을 민감하지 않은 mock data로 실제 `MenuBarDashboardView` SwiftUI UI에서 렌더링한 이미지로 교체했습니다.
- README에 번체 중국어, 간체 중국어, 일본어, 한국어, 프랑스어, 스페인어 메뉴바 현지화 스크린샷을 추가했습니다.
- 손으로 편집한 이미지가 아니라 App의 SwiftUI UI에서 README 메뉴바 스크린샷을 다시 생성할 수 있도록 gated 스크린샷 생성 테스트를 추가했습니다.

## 참고 사항

- 이 prerelease는 v1.0.14-rc.18의 App runtime 기능 세트를 유지하며, 수정된 문서와 release assets를 검증하기 위해 build number를 올립니다.
- 스크린샷 mock data는 민감하지 않은 데이터이며 실제 계정을 포함하지 않습니다.
