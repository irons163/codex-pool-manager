# CodexPoolManager v1.0.14-rc.19

Fecha de lanzamiento: 2026-07-03

## Mejoras

- Se reemplazó la captura de menu bar del README por una imagen renderizada desde la vista SwiftUI real `MenuBarDashboardView` con datos mock no sensibles.
- Se añadieron capturas localizadas de menu bar en el README para chino tradicional, chino simplificado, japonés, coreano, francés y español.
- Se añadió una prueba gated de generación de capturas para regenerar las imágenes de menu bar del README desde la UI SwiftUI de la app, sin edición manual de imágenes.

## Notas

- Esta prerelease conserva el conjunto de funciones runtime de v1.0.14-rc.18 e incrementa el build number para validar los assets corregidos de documentación/release.
- Los datos mock de las capturas no son sensibles y no incluyen cuentas reales.
