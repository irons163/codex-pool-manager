# CodexPoolManager v1.0.15

リリース日：2026-07-03

## 主な変更

- アカウントカードに、利用可能なリセット枠と各リセット枠の推定期限を表示するようにしました。
- Minimal アカウントカードでは、`リセット 2回 · 7/30, 8/1` のような短い日付表示でまとめます。
- メニューバーダッシュボードで、アカウントグループ切り替え、コンパクトな警告 popover、Plus/Pro バッジ、リセット枠期限の詳細、常時表示の切り替えボタンをサポートしました。
- version/build 更新後にローカライズされた「新機能」案内を表示し、Settings から最新機能ノートを再表示できるようにしました。
- OAuth アカウントの自動更新安定性を改善し、すべての対応言語で release notes を提供しました。
- README のメニューバー画像は、非機密の mock data を使って実際の SwiftUI メニューバーダッシュボードからレンダリングしたものになりました。

## 注意事項

- リセット枠の期限は前回の同期成功時刻に 30 日を加えて推定しているため、実際の期限とは異なる場合があります。
- この stable release は、v1.0.14 rc.16 から rc.19 の prerelease サイクルで検証済みの変更をまとめたものです。
