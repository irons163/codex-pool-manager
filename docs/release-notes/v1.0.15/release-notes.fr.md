# CodexPoolManager v1.0.15

Date de publication : 2026-07-03

## Nouveautés

- Les cartes de compte affichent désormais les crédits de réinitialisation disponibles et l'expiration estimée de chaque crédit.
- Les cartes Minimal résument les crédits avec des dates compactes comme `2 réinitialisations · 7/30, 8/1`.
- Le tableau de bord de la barre de menus prend maintenant en charge le filtrage par groupe de comptes, les popovers d'avertissement compacts, les badges Plus/Pro, les détails d'expiration des crédits et les boutons de bascule toujours visibles.
- Ajout de prompts Nouveautés localisés après les changements de version/build, avec une action dans Settings pour rouvrir les dernières notes.
- Amélioration de la stabilité du renouvellement automatique OAuth et publication des release notes dans toutes les langues prises en charge.
- Les captures menu bar du README sont maintenant rendues depuis le vrai tableau de bord SwiftUI avec des données mock non sensibles.

## Notes

- L'expiration des crédits est estimée à partir de la dernière synchronisation réussie plus 30 jours ; l'expiration réelle peut différer.
- Cette version stable regroupe les changements validés pendant le cycle prerelease v1.0.14 rc.16 à rc.19.
