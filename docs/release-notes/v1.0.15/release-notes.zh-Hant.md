# CodexPoolManager v1.0.15

發布日期：2026-07-03

## 重點更新

- 帳號卡現在會顯示可重置額度，以及每一次可重置額度的推估期限。
- 極簡帳號卡會用 `可重置 2 次 · 7/30, 8/1` 這類精簡日期文字顯示。
- Menu bar 儀表板現在支援帳號群組切換、精簡警告 popover、Plus/Pro 標籤、可重置期限細節，以及永遠可見的切換按鈕。
- 版本或 build 更新後會顯示本地化「新功能」提示，也可從設定重新查看最新功能說明。
- 改善 OAuth 帳號自動續期穩定性，並補齊所有支援語系的 release notes。
- README 的 menu bar 截圖現在由真正的 SwiftUI menu bar 儀表板搭配非敏感 mock data 渲染產出。

## 注意事項

- 可重置額度期限是依前次成功同步時間加 30 天推估，實際期限可能不同。
- 此 stable release 彙整 v1.0.14 rc.16 到 rc.19 prerelease 週期中已驗證的變更。
