# CodexPoolManager v1.0.15

发布日期：2026-07-03

## 重点更新

- 账号卡现在会显示可重置额度，以及每一次可重置额度的推估期限。
- 极简账号卡会用 `可重置 2 次 · 7/30, 8/1` 这类精简日期文字显示。
- Menu bar 仪表板现在支持账号群组切换、精简警告 popover、Plus/Pro 标签、可重置期限细节，以及始终可见的切换按钮。
- 版本或 build 更新后会显示本地化“新功能”提示，也可从设置重新查看最新功能说明。
- 改善 OAuth 账号自动续期稳定性，并补齐所有支持语言的 release notes。
- README 的 menu bar 截图现在由真正的 SwiftUI menu bar 仪表板搭配非敏感 mock data 渲染生成。

## 注意事项

- 可重置额度期限是以前次成功同步时间加 30 天推估，实际期限可能不同。
- 此 stable release 汇总 v1.0.14 rc.16 到 rc.19 prerelease 周期中已验证的变更。
