# CodexPoolManager v1.0.15

릴리스 날짜: 2026-07-03

## 주요 변경 사항

- 계정 카드에 사용 가능한 초기화 횟수와 각 초기화의 추정 만료 시간이 표시됩니다.
- Minimal 계정 카드는 `초기화 2회 · 7/30, 8/1`처럼 짧은 날짜 형식으로 요약합니다.
- 메뉴바 대시보드에 계정 그룹 전환, 간결한 경고 popover, Plus/Pro 배지, 초기화 만료 상세 정보, 항상 표시되는 전환 버튼을 추가했습니다.
- version/build 업데이트 후 현지화된 새로운 기능 안내를 표시하며, Settings에서 최신 기능 노트를 다시 열 수 있습니다.
- OAuth 계정 자동 갱신 안정성을 개선했고, 모든 지원 언어의 release notes를 제공합니다.
- README 메뉴바 스크린샷은 민감하지 않은 mock data로 실제 SwiftUI 메뉴바 대시보드에서 렌더링한 이미지입니다.

## 참고 사항

- 초기화 만료 시간은 이전 성공 동기화 시각에 30일을 더해 추정하므로 실제 만료 시간과 다를 수 있습니다.
- 이 stable release는 v1.0.14 rc.16부터 rc.19 prerelease 주기에서 검증된 변경 사항을 묶은 버전입니다.
