# CodexPoolManager v1.0.15-rc.1

发布日期：2026-07-11

## 改进

- 重置额度到期日期现在直接使用 Codex 账号详情端点返回的 `expires_at`，不再根据上次成功同步时间进行估算。
- 账号卡片和重置提醒会在数据可用时显示 API 提供的准确日期。
- API 提供的日期不再显示“估算日期”警告；仅在必须使用备用估算时才显示警告。
- 账号快照会保留到期日期来源，重新启动后仍能区分 API 准确日期与备用估算日期。
- README 与所有支持的界面语言已同步更新，一致说明 API 准确日期与备用估算。

## 兼容性

- 无法获取账号详情或到期日期时，现有估算机制仍会作为备用方案。
- 旧版本创建的快照仍可读取，并会保守地视为估算数据。

## Prerelease 说明

- 此 prerelease 用于验证重置额度准确到期日期同步及数据不可用时的备用行为，再进入下一次 stable 发布。
