# CodexPoolManager v1.0.15-rc.1

출시일: 2026-07-11

## 개선 사항

- 재설정 크레딧 만료일은 더 이상 마지막 동기화 시각으로 추정하지 않고 Codex 계정 세부 정보 엔드포인트가 반환한 `expires_at` 값을 직접 사용합니다.
- 계정 카드와 재설정 알림은 데이터가 있을 때 API가 제공한 정확한 날짜를 표시합니다.
- API에서 받은 날짜에는 예상 날짜 경고를 표시하지 않으며, 대체 추정값을 사용해야 할 때만 경고를 표시합니다.
- 만료일 출처를 계정 스냅샷에 보존하여 앱을 다시 실행한 뒤에도 정확한 API 날짜와 추정 날짜를 구분합니다.
- README와 지원되는 모든 UI 언어를 업데이트하여 정확한 API 날짜와 대체 추정값을 일관되게 설명합니다.

## 호환성

- 계정 세부 정보나 만료일을 가져올 수 없으면 기존 추정 방식을 대체 수단으로 계속 사용합니다.
- 이전 버전에서 만든 스냅샷도 계속 읽을 수 있으며 안전을 위해 추정 데이터로 처리합니다.

## Prerelease 참고

- 이번 prerelease에서는 다음 stable 배포 전에 정확한 만료일 동기화와 데이터 미제공 시 대체 동작을 검증합니다.
