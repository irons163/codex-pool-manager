# CodexPoolManager v1.0.14-rc.17

Fecha de lanzamiento: 2026-07-01

## Mejoras

- Se reorganizó el panel de la barra de menús como una lista de cuentas única y compacta con filtro por grupo.
- La lista de cuentas de la barra de menús ahora respeta los ajustes de ordenación de cuentas de la ventana principal.
- Se añadieron insignias Plus / Pro a las filas de cuentas en la barra de menús.
- La cuenta activa conserva la marca azul y también muestra el botón Cambiar.
- Las advertencias de cuenta y las notas de estimación de restablecimientos se movieron a botones compactos de exclamación.
- Cada fila de cuenta muestra los restablecimientos disponibles y el vencimiento estimado de cada restablecimiento.
- El diseño compacto mantiene y alinea las horas exactas de restablecimiento weekly y 5h.

## Notas

- El vencimiento de los restablecimientos se estima sumando 30 días a la sincronización correcta anterior; el vencimiento real puede variar.
- Esta prerelease sirve para validar el nuevo flujo de cuentas en la barra de menús y la visualización de vencimientos de restablecimientos antes del lanzamiento stable.
