# CodexPoolManager v1.0.14-rc.17

Release date: 2026-07-01

## Improvements

- Reworked the menu bar dashboard into a compact single account list with group filtering.
- Mirrored the main dashboard account ordering settings in the menu bar list.
- Added Plus/Pro plan badges to menu bar account rows.
- Kept the Switch button visible for the active account while preserving the active checkmark.
- Moved account warnings and reset-credit estimation notes behind compact exclamation buttons.
- Added reset-credit availability and per-reset estimated expiry details directly on each account row.
- Kept weekly and 5-hour usage reset times precise and aligned in the compact row layout.

## Notes

- Reset-credit expiry is estimated from the previous successful sync plus 30 days; the actual expiry may differ.
- This prerelease is intended to validate the redesigned menu bar account workflow and reset-credit expiry display before a stable rollout.
