# CodexPoolManager v1.0.14-rc.17

發布日期：2026-07-01

## 改進

- 將 menu bar 儀表板整理成更精簡的單一帳號清單，並加入群組切換。
- menu bar 帳號排序會套用主視窗的帳號排序設定。
- menu bar 帳號列新增 Plus / Pro 方案標籤。
- 當前帳號保留藍色勾勾，同時也會顯示「切換」按鈕。
- 帳號警示與可重置額度推估說明改藏在精簡的驚嘆號按鈕中。
- 帳號列直接顯示可重置次數，以及每一次重置額度的推估到期時間。
- 精簡列中保留並對齊 weekly 與 5h 的精確重置時間。

## 注意事項

- 可重置額度期限是依前次成功同步時間加 30 天推估，實際期限可能不同。
- 此 prerelease 用於正式發布前驗證新版 menu bar 帳號操作流程與可重置額度到期顯示。
