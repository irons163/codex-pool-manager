# CodexPoolManager v1.0.14-rc.17

发布日期：2026-07-01

## 改进

- 将 menu bar 仪表板整理成更精简的单一账号列表，并加入群组切换。
- menu bar 账号排序会套用主窗口的账号排序设置。
- menu bar 账号行新增 Plus / Pro 方案标签。
- 当前账号保留蓝色勾选，同时也会显示「切换」按钮。
- 账号警示与可重置额度估算说明改藏在精简的感叹号按钮中。
- 账号行直接显示可重置次数，以及每一次重置额度的预计到期时间。
- 精简行中保留并对齐 weekly 与 5h 的精确重置时间。

## 注意事项

- 可重置额度期限是根据上次成功同步时间加 30 天估算，实际期限可能不同。
- 此 prerelease 用于正式发布前验证新版 menu bar 账号操作流程与可重置额度到期显示。
