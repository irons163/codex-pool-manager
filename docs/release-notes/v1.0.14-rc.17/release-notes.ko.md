# CodexPoolManager v1.0.14-rc.17

릴리스 날짜: 2026-07-01

## 개선 사항

- 메뉴 바 대시보드를 그룹 전환이 있는 더 간결한 단일 계정 목록으로 정리했습니다.
- 메뉴 바 계정 목록에 메인 창의 계정 정렬 설정을 반영하도록 했습니다.
- 메뉴 바 계정 행에 Plus / Pro 플랜 배지를 추가했습니다.
- 현재 계정에는 파란 체크 표시를 유지하면서도 “전환” 버튼을 함께 표시합니다.
- 계정 경고와 초기화 크레딧 추정 설명을 간결한 느낌표 버튼 안으로 이동했습니다.
- 계정 행에 사용 가능한 초기화 횟수와 각 초기화 크레딧의 예상 만료 시간을 직접 표시합니다.
- 간결한 행 레이아웃에서도 weekly 및 5h의 정확한 초기화 시간이 유지되고 정렬됩니다.

## 참고

- 초기화 크레딧 만료는 이전 동기화 성공 시각에 30일을 더해 추정하므로 실제 만료 시각과 다를 수 있습니다.
- 이 prerelease는 stable 배포 전에 새 메뉴 바 계정 흐름과 초기화 크레딧 만료 표시를 검증하기 위한 버전입니다.
