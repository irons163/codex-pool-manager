# CodexPoolManager v1.0.14-rc.17

リリース日：2026-07-01

## 改善

- メニューバーダッシュボードを、グループ切り替え付きのコンパクトな単一アカウント一覧に整理しました。
- メニューバーのアカウント並び順に、メイン画面のアカウント並び順設定を反映するようにしました。
- メニューバーのアカウント行に Plus / Pro プランバッジを追加しました。
- 現在のアカウントでは青いチェックを残しつつ、「切り替え」ボタンも表示するようにしました。
- アカウント警告とリセット枠の推定説明を、コンパクトな感嘆符ボタン内に移動しました。
- アカウント行に、利用可能なリセット回数と各リセット枠の推定期限を直接表示するようにしました。
- コンパクトな行レイアウトで weekly と 5h の正確なリセット時刻を保持し、揃えて表示します。

## 注意事項

- リセット枠の期限は前回の同期成功時刻に 30 日を加えて推定しているため、実際の期限とは異なる場合があります。
- この prerelease は、stable 公開前に新しいメニューバーのアカウント操作とリセット枠期限表示を検証するためのものです。
