# CodexPoolManager v1.0.16

リリース日：2026-07-11

## ハイライト

- リセットクレジットの有効期限は、前回の同期時刻から推定せず、Codex アカウント詳細 API が返す正確な `expires_at` を直接使用するようになりました。
- アカウントカード、メニューバーダッシュボード、リセット通知には、取得できる場合は API の正確な日付が表示されます。
- API から取得した日付には推定日の警告を表示せず、フォールバック推定が必要な場合にのみ警告を表示します。
- 有効期限の取得元をアカウントスナップショットに保存し、再起動後も API の正確な日付と推定日を区別します。
- README と対応するすべての UI 言語を更新し、API の正確な日付とフォールバック推定を一貫して説明します。

## 信頼性と互換性

- アカウント詳細または有効期限を取得できない場合は、既存の推定処理を自動的にフォールバックとして使用します。
- 旧バージョンのスナップショットは引き続き読み込め、安全のため推定データとして扱われます。
- この正式版には v1.0.15-rc.1 で検証された変更が含まれます。
