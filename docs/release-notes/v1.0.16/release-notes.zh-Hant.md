# CodexPoolManager v1.0.16

發布日期：2026-07-11

## 重點更新

- 可重置額度到期日現在直接使用 Codex 帳號詳細資料 API 回傳的確切 `expires_at`，不再依上次成功同步時間推估。
- 帳號卡、menu bar 儀表板與重置提醒會在資料可用時顯示 API 提供的確切日期。
- API 提供的日期不再顯示「推估日期」警告；只有必須使用備援推估時才會顯示警告。
- 帳號快照會保留到期日來源，重新啟動後仍能區分 API 確切日期與推估日期。
- README 與所有支援的介面語系已同步更新，一致說明 API 確切日期與備援推估。

## 穩定性與相容性

- 無法取得帳號詳細資料或到期日時，既有推估機制會自動作為備援。
- 舊版本建立的快照仍可讀取，並會保守地視為推估資料。
- 此正式版彙整 v1.0.15-rc.1 已驗證的變更。
