# CodexPoolManager v1.0.16

发布日期：2026-07-11

## 重点更新

- 重置额度到期日期现在直接使用 Codex 账号详情 API 返回的准确 `expires_at`，不再根据上次成功同步时间进行估算。
- 账号卡片、菜单栏仪表板和重置提醒会在数据可用时显示 API 提供的准确日期。
- API 提供的日期不再显示“估算日期”警告；仅在必须使用备用估算时才显示警告。
- 账号快照会保留到期日期来源，重新启动后仍能区分 API 准确日期与估算日期。
- README 与所有支持的界面语言已同步更新，一致说明 API 准确日期与备用估算。

## 稳定性与兼容性

- 无法获取账号详情或到期日期时，现有估算机制会自动作为备用方案。
- 旧版本创建的快照仍可读取，并会保守地视为估算数据。
- 此正式版汇总了 v1.0.15-rc.1 中已验证的变更。
